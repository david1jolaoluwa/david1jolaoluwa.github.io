How to use this system
1️⃣
Set up your Google Sheet
Open the AimingHigh_StudentTracker.xlsx file you downloaded and import it into Google Sheets. Then go to File → Share → Publish to web, publish the whole document, and copy the Sheet ID from your URL.
2️⃣
Add your Sheet ID to config.js
Open js/config.js and replace YOUR_SHEET_ID_HERE with the ID from your Google Sheet URL. Push to GitHub and the site will update instantly.
3️⃣
Add a student
Duplicate the 👤 STUDENT TEMPLATE tab in your Sheet. Rename it with the student's name (e.g. Amira). Then create a matching HTML file in the students/ folder by copying template.html and renaming it amira.html. The page auto-reads the sheet tab with the matching name.
4️⃣
Share with parents
Send parents the link to their child's page — e.g. david1jolaoluwa.github.io/students/amira. The page updates live every time you update the Google Sheet. No login needed for parents.
